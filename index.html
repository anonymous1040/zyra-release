How to install Zyra Chat

Download the APK
Tap the Download Zyra Chat button on the previous screen. The file will save to your phone's Downloads folder.
2
Enable installs from unknown sources
Go to Settings → Apps → Special access → Install unknown apps and allow the browser or file manager you're using.

Use your file manager or the browser's download bar to open zyra-chat.apk. Tap Install and wait. If prompted, accept permissions.
